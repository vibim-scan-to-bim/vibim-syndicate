Scan to BIM is the process of converting a physical building or space into a precise Building Information Model using 3D laser scanning technology.

How it works

A LiDAR scanner captures the site as a point cloud — a dataset containing millions of 3D coordinates representing every surface it can reach. Technicians register multiple scan positions into a unified point cloud, then model it in Autodesk Revit or similar BIM authoring tools to produce a structured, object-based as-built model.

Common applications

  • Renovation and adaptive reuse
  • MEP clash detection in existing buildings
  • Historic preservation documentation
  • Facilities management digital twins

Deliverables

A standard Scan to BIM project produces a native Revit model (or IFC export) with point cloud as reference, a QA deviation report, and scan data in .rcp / .e57 format. Typical accuracy is +/-6mm at LOD 200-350.
